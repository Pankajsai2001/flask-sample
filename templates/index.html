What is it?

  • This 'Application Rationalization Framework' is a solution to assess ARF Capability completion and maturity for a Telecom Service Provider.
  • The framework is built using experience from other Telecom Service providers across the globe, and best practices in ARF from Telecom and other industries.
  • Using this framework, Telecom Service Provider can identify any gaps in existing ARF Capabilities, and can work on contextual recommendations to work on gaps, and to create a requirements backlog.


How to Use it?

  • Go through the questionnare and answer the form for functionalities, and the framework will provide an output with scores, and missing functionalties.
